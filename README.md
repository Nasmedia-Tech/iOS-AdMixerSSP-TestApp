# iOS-AdMixerSSP-TestApp
케이티 나스미디어 **AdMixerMediation SDK** 샘플 앱입니다.  
앱을 실행하기 전에 아래 항목들을 설정해주세요.

<br>

## 설정 방법

1. **GADIdentifiers**  
   - `Info.plist` 파일에서 설정합니다.

2. **MediaKey 및 AdUnitId**  
   - `Constants` 파일에서 설정합니다.

<br>

## 사용 가이드

1. pod update를 실행해주세요. 
2. 발급받은 **GADIdentifier**, **mediaKey**, 그리고 **adunitId** 값을 준비하세요.  
3. 위에서 설명한 위치에 각각의 값을 기입하세요.
5. 앱을 실행하여 테스트를 진행하세요.

자세한 내용은 가이드를 참고해주세요.
